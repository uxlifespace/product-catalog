Політика конфіденційності

Останнє оновлення: 08.07.2026

Ця Політика конфіденційності визначає порядок збору, використання та захисту персональних даних користувачів сайту marcelco.com.ua та Instagram-акаунту бренду Marcel&Co (далі — "Сайт"), що належать ФОП Уфатову Дмитру Сергійовичу (далі — "Оператор").

Оператор дотримується вимог Закону України "Про захист персональних даних" №2297-VI та чинного законодавства України у сфері електронної комерції.

1. Які дані ми збираємо

При оформленні замовлення Оператор може збирати такі дані: ім'я та прізвище, номер телефону, адресу електронної пошти, адресу доставки (відділення Нової пошти), інформацію про замовлення та історію покупок.

2. Мета обробки даних

Персональні дані обробляються з метою: оформлення та виконання замовлень, зв'язку з Покупцем щодо статусу замовлення, здійснення доставки товару, обробки платежів, надання підтримки клієнтам, покращення якості сервісу.

3. Передача даних третім особам

Для виконання замовлення персональні дані можуть передаватися таким контрагентам:

— Компанії KeyCRM (система обліку замовлень) — для обробки та ведення замовлень;
— Перевізнику "Нова пошта" — для здійснення доставки;
— Monobank Acquiring — для обробки онлайн-платежів карткою;
— Google Analytics — для аналізу відвідуваності сайту в знеособленому вигляді.

Оператор не продає та не передає персональні дані третім особам з маркетинговою метою без згоди користувача.

4. Оплата карткою

Онлайн-оплата здійснюється через платіжну систему Monobank Acquiring. Дані банківської картки Покупця не зберігаються та не обробляються Оператором — вони передаються безпосередньо до захищеної платіжної системи Monobank відповідно до стандарту PCI DSS.

5. Строк зберігання даних

Персональні дані зберігаються протягом строку, необхідного для виконання договірних зобов'язань, а також протягом строку, передбаченого законодавством України для ведення бухгалтерського та податкового обліку.

6. Права користувача

Користувач має право: отримати інформацію про свої персональні дані, вимагати виправлення неточних даних, вимагати видалення своїх персональних даних (за умови відсутності законних підстав для їх подальшого зберігання), відкликати згоду на обробку даних.

Для реалізації цих прав звертайтесь на email: support@marcelco.com.ua

7. Файли cookie

Сайт може використовувати файли cookie для аналітики відвідуваності та покращення роботи сайту. Користувач може відключити cookie в налаштуваннях свого браузера.

8. Захист даних

Оператор вживає технічних та організаційних заходів для захисту персональних даних від несанкціонованого доступу, втрати чи розголошення.
