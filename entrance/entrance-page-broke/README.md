# entrance-page

QR-лендинг «Entrance» для бренду **Marcel&Co** (ласощі для собак).
Один статичний файл `index.html` (HTML + CSS + vanilla JS, без фреймворків і збірки).
Усі ілюстрації вбудовані inline як SVG.

## Структура
- `index.html` — сторінка (готова до деплою).
- `assets/` — вихідні SVG (логотип, написи, іконки) для довідки.
- `api/lead.js` — serverless-функція: приймає номер і створює заявку у воронці KeyCRM.
- `docs/` — інструкція з публікації, домену та інтеграцій.
- `.env.example` — шаблон змінних оточення (реальні ключі НЕ комітимо).

## Локальний перегляд
Відкрити `index.html` у браузері — більше нічого не потрібно.

## Деплой
1. Залити на Netlify / Cloudflare Pages / Vercel (статичний сайт).
2. Підв'язати домен через DNS (A / CNAME) + увімкнути HTTPS.
3. Кнопки: замінити `href` на реальні (каталог, Instagram, Telegram).
4. KeyCRM: задеплоїти `api/lead.js`, прописати `KEYCRM_API_KEY` та `KEYCRM_PIPELINE_ID`
   у Environment Variables хостингу, форму спрямувати на `/api/lead`.

Деталі — у `docs/Інструкція_публікація_і_інтеграції.md`.
